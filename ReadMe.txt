###Welcome to Hangman made by Aleksandar Gjorgev.

To start program:
1. Click on Hangman.exe
2. Enjoy

###Rules for playing:
1. You can only enter 1 character per input
2. No numbers
3. No empty inputs

###How to change words for the game:
1. Go inside Words_text.txt
2. Change words, but don't put no spaces! (only commas to seperate the words)
3. Save file
